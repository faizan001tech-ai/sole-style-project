# Cloudinary Image Upload Integration Guide

## Overview
Cloudinary is used for storing and serving product images and user avatars in production.

## Setup

### 1. Create Cloudinary Account
1. Go to https://cloudinary.com/
2. Sign up for free account (25GB storage, 25K transformations/month)
3. Go to Dashboard to get credentials:
   - Cloud Name
   - API Key
   - API Secret

### 2. Backend Configuration

#### Install Dependencies
```bash
cd backend
npm install cloudinary multer multer-storage-cloudinary
```

#### Create Config File
`backend/config/cloudinary.js`:
```javascript
import { v2 as cloudinary } from 'cloudinary';
import { CloudinaryStorage } from 'multer-storage-cloudinary';

cloudinary.config({
  cloud_name: process.env.CLOUDINARY_CLOUD_NAME,
  api_key: process.env.CLOUDINARY_API_KEY,
  api_secret: process.env.CLOUDINARY_API_SECRET,
});

// Storage for product images
const productStorage = new CloudinaryStorage({
  cloudinary: cloudinary,
  params: {
    folder: 'windsurf-gear/products',
    allowed_formats: ['jpg', 'jpeg', 'png', 'webp'],
    transformation: [{ width: 800, height: 800, crop: 'limit' }],
  },
});

// Storage for user avatars
const avatarStorage = new CloudinaryStorage({
  cloudinary: cloudinary,
  params: {
    folder: 'windsurf-gear/avatars',
    allowed_formats: ['jpg', 'jpeg', 'png', 'webp'],
    transformation: [{ width: 400, height: 400, crop: 'fill', gravity: 'face' }],
  },
});

export { cloudinary, productStorage, avatarStorage };
```

#### Environment Variables
Add to `backend/.env`:
```
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```

#### Create Upload Routes
`backend/routes/upload.js`:
```javascript
import express from 'express';
import multer from 'multer';
import { productStorage, avatarStorage } from '../config/cloudinary.js';
import { protect } from '../middleware/auth.js';

const router = express.Router();

const productUpload = multer({ storage: productStorage });
const avatarUpload = multer({ storage: avatarStorage });

// Upload product image (Admin only)
router.post('/product', protect, productUpload.single('image'), (req, res) => {
  try {
    if (!req.file) {
      return res.status(400).json({ message: 'No image uploaded' });
    }
    res.json({
      success: true,
      imageUrl: req.file.path,  // Cloudinary URL
      publicId: req.file.filename,
    });
  } catch (error) {
    res.status(500).json({ message: error.message });
  }
});

// Upload user avatar
router.post('/avatar', protect, avatarUpload.single('avatar'), (req, res) => {
  try {
    if (!req.file) {
      return res.status(400).json({ message: 'No avatar uploaded' });
    }
    res.json({
      success: true,
      avatarUrl: req.file.path,
      publicId: req.file.filename,
    });
  } catch (error) {
    res.status(500).json({ message: error.message });
  }
});

export default router;
```

#### Register Route in Server
`backend/server.js`:

### 3. Frontend Integration

#### Image Upload Component
`frontend/src/components/ImageUpload.jsx`:
```jsx
import { useState } from 'react';
import axios from 'axios';
import { Upload, X } from 'lucide-react';
import toast from 'react-hot-toast';

const ImageUpload = ({ onUpload, type = 'product' }) => {
  const [loading, setLoading] = useState(false);
  const [preview, setPreview] = useState(null);

  const handleFileSelect = async (e) => {
    const file = e.target.files[0];
    if (!file) return;

    // Preview
    const reader = new FileReader();
    reader.onloadend = () => setPreview(reader.result);
    reader.readAsDataURL(file);

    // Upload
    setLoading(true);
    const formData = new FormData();
    formData.append(type === 'avatar' ? 'avatar' : 'image', file);

    try {
      const token = localStorage.getItem('token');
      const response = await axios.post(
        `${import.meta.env.VITE_API_URL}/upload/${type}`,
        formData,
        {
          headers: {
            'Content-Type': 'multipart/form-data',
            Authorization: `Bearer ${token}`,
          },
        }
      );

      const imageUrl = type === 'avatar' 
        ? response.data.avatarUrl 
        : response.data.imageUrl;
      
      onUpload(imageUrl);
      toast.success('Image uploaded successfully!');
    } catch (error) {
      toast.error('Failed to upload image');
    } finally {
      setLoading(false);
    }
  };

  const handleRemove = () => {
    setPreview(null);
    onUpload('');
  };

  return (
    <div className="space-y-4">
      {preview && (
        <div className="relative inline-block">
          <img
            src={preview}
            alt="Preview"
            className="w-32 h-32 object-cover rounded-lg"
          />
          <button
            onClick={handleRemove}
            className="absolute -top-2 -right-2 bg-red-500 text-white rounded-full p-1"
          >
            <X className="w-4 h-4" />
          </button>
        </div>
      )}

      <label className="flex items-center justify-center w-32 h-32 border-2 border-dashed border-gray-300 rounded-lg cursor-pointer hover:border-primary-500">
        <input
          type="file"
          accept="image/*"
          onChange={handleFileSelect}
          className="hidden"
          disabled={loading}
        />
        {loading ? (
          <div className="animate-spin rounded-full h-6 w-6 border-b-2 border-primary-600" />
        ) : (
          <Upload className="w-6 h-6 text-gray-400" />
        )}
      </label>
    </div>
  );
};

export default ImageUpload;
```

### 4. Product Form with Image Upload
`frontend/src/components/ProductForm.jsx`:
```jsx
import { useState } from 'react';
import ImageUpload from './ImageUpload';

const ProductForm = () => {
  const [formData, setFormData] = useState({
    name: '',
    price: '',
    image: '',
    // ... other fields
  });

  const handleImageUpload = (imageUrl) => {
    setFormData({ ...formData, image: imageUrl });
  };

  return (
    <form>
      {/* Other fields */}
      <div>
        <label>Product Image</label>
        <ImageUpload onUpload={handleImageUpload} type="product" />
        {formData.image && (
          <img src={formData.image} alt="Product" className="mt-2 w-32 h-32 object-cover" />
        )}
      </div>
      {/* Submit button */}
    </form>
  );
};
```

## Image Optimization Tips

### 1. Auto-Optimization
Cloudinary automatically optimizes images by:
- Compressing to WebP when browser supports it
- Resizing based on device pixel ratio
- Lazy loading support

### 2. URL Transformations
Modify image URL for different sizes:
```javascript
// Original URL
const original = 'https://res.cloudinary.com/demo/image/upload/v1234567890/sample.jpg';

// Resized to 300x300
const thumbnail = original.replace('/upload/', '/upload/w_300,h_300,c_fill/');

// With quality optimization
const optimized = original.replace('/upload/', '/upload/q_auto,w_800/');
```

### 3. Responsive Images
```jsx
<img
  src={imageUrl.replace('/upload/', '/upload/w_400/')}
  srcSet={`
    ${imageUrl.replace('/upload/', '/upload/w_400/')} 400w,
    ${imageUrl.replace('/upload/', '/upload/w_800/')} 800w,
    ${imageUrl.replace('/upload/', '/upload/w_1200/')} 1200w
  `}
  sizes="(max-width: 600px) 400px, (max-width: 1000px) 800px, 1200px"
  alt="Product"
/>
```

## Troubleshooting

### Upload Fails
1. Check file size (max 10MB for free tier)
2. Verify file format (jpg, jpeg, png, webp)
3. Check Cloudinary dashboard for errors

### Images Not Displaying
1. Check URL is correct in database
2. Verify image exists in Cloudinary Media Library
3. Check browser console for 404 errors

### CORS Issues
Add to Cloudinary settings:
1. Go to Settings → Security
2. Add your domain to "Allowed fetch domains"

## Migration from Local to Cloudinary

If you have existing local images:

```javascript
// Migration script
import fs from 'fs';
import { cloudinary } from './config/cloudinary.js';
import Product from './models/Product.js';

const migrateImages = async () => {
  const products = await Product.find({ image: { $regex: '^/uploads/' } });
  
  for (const product of products) {
    const localPath = `./backend${product.image}`;
    
    try {
      const result = await cloudinary.uploader.upload(localPath, {
        folder: 'windsurf-gear/products',
      });
      
      product.image = result.secure_url;
      await product.save();
      
      console.log(`Migrated: ${product.name}`);
    } catch (error) {
      console.error(`Failed: ${product.name}`, error);
    }
  }
};
```

## Alternative: AWS S3

If you prefer AWS S3 over Cloudinary:

- Cheaper storage for large volumes
- Direct CDN integration with CloudFront
- More granular permissions
